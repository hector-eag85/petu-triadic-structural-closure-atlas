# DATA_AVAILABILITY

All datasets used in this study are public, reproducible from public sources, or accessible through public scientific repositories, databases or software packages.

Raw data are redistributed only where permitted by the original data source license or repository policy. Where raw datasets are too large to redistribute or governed by third-party access policies, download instructions and preprocessing scripts should be used instead of redistributing raw data.

Included in this package:

- Supplementary Tables ST0-ST15
- figure source tables
- processed outputs/logs where available
- audit files
