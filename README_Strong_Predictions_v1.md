# PETU Strong Predictions and Preregistered Tests v1

This file documents the preregistration-oriented strong-prediction framework associated with the PETU empirical atlas and the formal mathematical companion.

The strong-prediction document translates the PETU framework into explicit empirical test classes, including baseline hierarchies, null-model requirements, falsification criteria, reporting templates, domain templates, preregistered thresholds, and law-candidate criteria.

## Scope

The strong-prediction framework is intended to support future independent replication, adversarial testing, preregistered domain extensions, alternative operationalizations, and stronger falsification attempts.

It does not replace the empirical atlas and does not claim that PETU is already a completed universal law or a theory of everything.

## Claim status

PETU is presented as a candidate cross-domain empirical principle.

The strong-prediction framework defines testable criteria under which the PETU framework can be supported, weakened, revised, or falsified in future work.
